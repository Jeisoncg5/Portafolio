# Portafolio Web - Jeison Cristancho

Portafolio personal desarrollado con `HTML`, `CSS` y `JavaScript`, con una estética inspirada en videojuegos retro y una presentación enfocada en mostrar mi perfil profesional, habilidades técnicas, proyectos destacados y canales de contacto.

## Vista general

Este proyecto funciona como una carta de presentación digital. La interfaz combina un estilo visual tipo arcade con una navegación clara y animaciones suaves para destacar la información más importante del perfil.


## Tecnologías utilizadas

- `HTML5`
- `CSS3`
- `JavaScript`
- `Google Fonts`
